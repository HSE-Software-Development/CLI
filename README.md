## Запуск под Linux
``` bash
# build
chmod +x scripts/build.sh
./scripts/build.sh

# Run
chmod +x scripts/run.sh
./scripts/run.sh
```

scripts\build.bat

#Run
#scripts\run.bat
.\bin\cli-app.exe
